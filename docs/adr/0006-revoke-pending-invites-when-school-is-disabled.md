# Revoke Pending Invites When School Is Disabled

เมื่อ system admin ปิดใช้งานโรงเรียน ระบบต้องยกเลิกคำเชิญที่ยังไม่ถูกใช้งานของโรงเรียนนั้น และการกู้คืนโรงเรียนไม่ควรทำให้คำเชิญเดิมกลับมาใช้ได้อีก ผู้ดูแลต้องสร้างคำเชิญใหม่หลังโรงเรียนกลับมาใช้งาน เพื่อลดช่องทางเข้าระบบที่ค้างอยู่จากช่วงก่อนปิดใช้งาน
