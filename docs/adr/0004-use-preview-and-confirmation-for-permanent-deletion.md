# Use Preview And Confirmation For Permanent Deletion

การลบถาวรโดย system admin ไม่ต้องให้ผู้ใช้พิมพ์รหัสโรงเรียนหรือนักเรียนซ้ำ เพราะกลุ่ม system admin มีจำนวนน้อยและถูกเชิญโดยผู้ดูแลด้วยกันเอง แต่ต้องผ่านหน้า preview ที่โหลดข้อมูลสำเร็จ ต้องกรอกเหตุผลการจัดการข้อมูล และต้องยืนยันใน modal สุดท้ายก่อนลบ เพื่อป้องกันการลบจากปุ่มเดียวโดยไม่เห็นผลกระทบ
