# Block Login For Disabled Schools

เมื่อ system admin ปิดใช้งานโรงเรียน ผู้ใช้ที่ผูกกับโรงเรียนนั้นจะเข้าใช้งานระบบไม่ได้ แทนที่จะให้ login แล้วเห็นหน้าแจ้งเตือน การตัดสินใจนี้ลดความเสี่ยงที่ workflow ภายในหลุดไปแก้ไขข้อมูลของโรงเรียนที่ถูกปิดใช้งานแล้ว และทำให้สถานะข้อมูลเลิกใช้งานถูกบังคับใช้ตั้งแต่ขอบเขต auth
