# System Operations Center UI Boundary

ระบบจะเพิ่มศูนย์ดูแลระบบเป็นหน้าใหม่สำหรับ system admin เพื่อรวมงานจัดการผู้ใช้งานและงานจัดการข้อมูลไว้ในประสบการณ์เดียว แต่ยังแยก service/action ตาม domain เดิมเพื่อลดความเสี่ยงของ regression ด้านสิทธิ์และ auth หน้าเดิมอย่างศูนย์จัดการข้อมูลและจัดการผู้ใช้งานจะค่อยๆ ถูกโยงหรือ redirect เข้าหน้าใหม่นี้ แทนการย้าย logic ทั้งหมดเป็น module เดียวทันที
