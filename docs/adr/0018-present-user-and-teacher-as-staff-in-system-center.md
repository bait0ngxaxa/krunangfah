# Present User And Teacher As Staff In System Center

ศูนย์ดูแลระบบจะนำเสนอครูที่มีทั้งบัญชีผู้ใช้และโปรไฟล์ครูเป็นเป้าหมายเดียวใน UI ภายใต้แนวคิดบุคลากร แต่ action ภายใน detail จะแยกส่วนบัญชีผู้ใช้ เช่น email, role, delete/disable account ออกจากส่วนโปรไฟล์ครู เช่น ชื่อ โรงเรียน ห้องที่ปรึกษา และ project role เพื่อให้ system admin ค้นหาและเข้าใจเป็นคนเดียวกัน ขณะที่ audit และสิทธิ์ยังบอกได้ชัดว่าแก้ข้อมูลชนิดใด
