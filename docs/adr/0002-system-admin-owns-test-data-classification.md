# System Admin Owns Test Data Classification

เฉพาะ system admin เท่านั้นที่สามารถ mark หรือ unmark โรงเรียนและนักเรียนเป็นข้อมูลทดสอบได้ เพราะ flag นี้ส่งผลต่อความเสี่ยงของการลบถาวรและการแยกข้อมูลใช้งานจริงออกจากข้อมูลทดลอง ทุกการเปลี่ยนแปลงต้องมีเหตุผลการจัดการข้อมูลและถูกบันทึกในประวัติการจัดการข้อมูล
