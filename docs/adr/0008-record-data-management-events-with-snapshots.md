# Record Data Management Events With Snapshots

ประวัติการจัดการข้อมูลจะถูกเก็บเป็นตารางกลางของเหตุการณ์การจัดการข้อมูล พร้อม snapshot ของเป้าหมายและผลกระทบในเวลาที่ทำรายการ เพื่อให้ system admin ตรวจสอบย้อนหลังได้แม้โรงเรียนหรือนักเรียนถูกลบถาวรออกจากตารางหลักแล้ว รูปแบบนี้ดีกว่าการผูก log กับ record เป้าหมายโดยตรง เพราะ record เป้าหมายอาจไม่อยู่หลังการลบถาวร
